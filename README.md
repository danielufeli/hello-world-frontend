## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- Make sure you have installed nodejs

### Setup
- git clone git@github.com:danielufeli/hello-world-frontend.git

### Install
- run npm install
### Usage
- run npm start
### Run tests

- run npm test
